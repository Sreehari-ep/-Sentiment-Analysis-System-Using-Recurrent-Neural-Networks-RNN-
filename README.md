## 📌 Project Overview

This project is a **Deep Learning-based Sentiment Analysis System** that classifies IMDb movie reviews as **Positive** or **Negative** using a **Recurrent Neural Network (RNN)** built with **TensorFlow/Keras**.

The application processes user-input movie reviews, performs text preprocessing and tokenization, converts the text into numerical sequences, and predicts the sentiment with a confidence score.

A professional **Streamlit web application** provides an interactive interface for real-time sentiment prediction.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/IMDb-Sentiment-Analysis-RNN.git
```

Move into the project directory

```bash
cd IMDb-Sentiment-Analysis-RNN
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
